# WardPass 🔑🔒

WardPass is a password solutions application. It has **two (2)** parts: A password generator and a password manager. The password generator allows you to create strong and secure passwords, and the password manager securely stores those passwords.

With the password manager, you can easily organize your passwords into different folders, and copy and paste them into your clipboard when you want to sign in.

## How it works?

It's very simple. **Here are the instructions:**

### Password Generator

If you wish to generate a new password, click the "Generate Password" button. By default, it will generate a very strong and secure password. If you wish to customize your password, you can by specifying the length and selecting the characters you want to include.

### Password Manager

1. Sign up for the application, if you haven't already. When logged in, you will be presented with your own dashboard.
2. Start by creating a Vault (a folder), give it any name you like. When the vault is created, select the plus (+) button to add a new password.
3. A pop-up will open asking you to input the name, website link (optional), username, email, and password of the service. Input the necessary data.
4. When finished, click the "Add" button, and the password will be added and remembered.
5. That is basically it! You can add as much passwords as you like.
