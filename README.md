# WardPass 🔑🔒

WardPass is a secure password management solution. With WardPass, you can easily organize your passwords into different folders, and use them when you want to sign in to any service.

## How it works?

It's very simple. **Here are the instructions:**

1. Sign up for the application, if you haven't already. When logged in, you will be presented with your own dashboard.
2. Start by creating a Vault (a folder), give it any name you like. When the vault is created, select the plus (+) button to add a new password.
3. A pop-up will open asking you to input the name, website link (optional), username, email, and password of the service. Input the necessary data.
4. When finished, click the "Add" button, and the password will be added and remembered.
5. That is basically it! You can add as much passwords as you like.
6. In the future, you can use our chrome extension that will allow you to store and retrieve passwords for your services automatically.
